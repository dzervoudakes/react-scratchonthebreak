# Scratch On The Break
> Rebuilt my buddy's [Scratch On The Break](https://scratchonthebreak.com/) site with React.

- - - -

### Instructions

Start Local Server
```
npm start
```

Run Linting
```
npm run lint
```

Run Linting with Fix
```
npm run lint:fix
```

Build for Production
```
npm run build
```

- - - -

### Some Technologies Used
* React
* ES6 + Babel
* Webpack
* React-router-dom
* Material-ui
* Node + Express
